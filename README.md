# ⚔️ QuarrelsomeQuests
QuarrelsomeQuests is a deck-building card game revolving around your 'heros', cards that go into battle for you. You can collect, purchase and pull these cards in the store found ingame, or collect them from various ingame events! 


## How to Install
Because QuarrelsomeQuests is under development, there is no download package to play the game as a desktop aplication. But the game IS playable on github pages. There you can start your hunt for a perfect deck! 

Play on the GitHub Pages site here:
👉 https://volive-io.github.io/QuarrelsomeQuests/
